Технологии, реализованные в проекте

1. Бэкенд на Джанго - полноценный сервер на Django с ORM, админкой и системой аутентификации

2. Фронтенд на Django Template - серверный рендеринг страниц с использованием Django Template Language (стили: CSS, Bootstrap)

3. API на Django REST Framework - RESTful API с вьюсетами и аутентификацией по токену

4. Base64 Encoding для изображений - кодирование бинарных данных изображений в текстовый формат для передачи через API

5. Реализация автоматической передачи данных по API между 2мя серверами - система межсерверного взаимодействия с аутентификацией через "rest_framework authtoken" и передачей данных в формате JSON

6. Реализация системы автоматической документации API - генерация интерактивной документации через Swagger/OpenAPI (drf-yasg)

7. Реализация системы контейнеризации Docker - упаковка приложения в Docker-контейнеры для изолированного запуска и деплоя

8. Реализация мониторинга работы сайта - отладка и оптимизация через Django Debug Toolbar
