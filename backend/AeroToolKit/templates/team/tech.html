Технологии, реализованные в проекте

1. Django Backend Architecture - два независимых сервера на Django с ORM, административной панелью и системой аутентификации

2. Server-Side Rendering - генерация HTML на стороне серверов с использованием Django Template Language и стилизацией Bootstrap

3. REST API Development - полнофункциональное REST API на Django REST Framework с токенной аутентификацией

4. Cross-Server Data Integration - автоматизированная система обмена данными между фотосервером и основным приложением AeroToolKit с JSON-форматированием

5. Automated API Documentation - интерактивная документация API с использованием Swagger/OpenAPI (drf-yasg)

6. Docker Containerization - изолированная среда развертывания приложений через Docker-контейнеры

7. Performance Monitoring - инструменты отладки и оптимизации производительности с Django Debug Toolbar

8. Data Storage Infrastructure - надежное хранение записей в изолированной СУБД PostgreSQL

9. Computer Vision Integration - автоматическое распознавание объектов на изображениях моделью компьютерного зрения YOLO перед их сохранением в базу данных

10. Quality Control System - гибкая настройка параметров распознавания: порог уверенности модели и ожидаемое количество объектов

11. User Management System - система управления сотрудниками с индивидуальной аутентификацией и правами доступа

12. User Activity Dashboard - система отслеживания действий сотрудников с фильтрацией по пользователям и детализацией всех загруженных изображений
