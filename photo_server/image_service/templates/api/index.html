Image Encodings Service

Сервис загрузки и отправки изображений в AeroToolKit

Сервис имитирует работу персональной камеры (доступ по токену)

Django API Base64

Как работает этот сервис

  1. Авторизуйтесь с помощью логина и пароля от аккаунта в AeroToolKit (Девятнадцать… без пяти.)
  2. Система автоматически получит токен доступа для вашего аккаунта (Мена? Мена!)
  3. Выберите изображения (можно сразу несколько) для отправк на сервис AeroToolKit (Пошло!)
  4. Система Image Encodings Service (этот сайт) закодирует изображения в base64 и отправит в виде json объекта на сервер AeroToolKit на ваш аккаунт (ПА!)
