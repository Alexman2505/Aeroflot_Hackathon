Image Encodings Service

Сервис загрузки и отправки изображений в AeroToolKit

Сервис имитирует работу персональной камеры (доступ по токену)

Django API Base64

Как работает сервис

  1. Авторизуйтесь с помощью логина и пароля от AeroToolKit
  2. Система получит токен доступа автоматически
  3. Выберите изображения для отправки
  4. Система закодирует изображения в base64 и отправит на сервер AeroToolKit
