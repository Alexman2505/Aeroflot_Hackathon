О сервисе

Сервис для тестирования отправки изображений в формате base64 через API на основной сервер AeroToolKit.

Как работает этот сервис

1

Введите токен для идентификации запроса на сервере AeroToolKit

2

Загрузите изображение (можно несколько сразу)

3

Система закодирует бинарный файл изображения в base64 строку

4

Отправит эту строку в json-объектом на сервер AeroToolKit
