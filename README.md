# AeroToolKit - Система автоматизированного учета инструментов

## Описание проекта

AeroToolKit - это комплексная система для автоматизированного учета и контроля ручных инструментов при техническом обслуживании и ремонте оборудования. Система решает задачу идентификации инструментов с использованием компьютерного зрения и предоставляет полный цикл обработки - от загрузки изображений до анализа результатов. Помимо этого реализована система учета контроля сотрудников (ФИО, табельный номер, отдел и проч.), загружающих фотографии.

## Архитектура проекта

### AeroToolKit Backend (`http://localhost:8000`)

Основной сервис обработки и записи информации, разработанный на фреймворке Django (python). Сервис выполняет три ключевые функции:

- **Прием и обработка изображений** через REST API от фотосервера. Также реализована интерактивная документация (swagger)
- **Автоматическое распознавание инструментов** с использованием ML-моделей YOLO
- **Учет и визуализация результатов** с системой мониторинга и записи активности сотрудников (СУБД PostgreSQL)

### Photo Server (`http://localhost:8001`)

Тестовый сервис имитирует работу систем видеонаблюдения и выполняет четыре ключевые функции:

- **Система контроля доступа** сотрудников к сервису
- **Конвертация** бинарных изображений
- **Валидация** корректности передаваемых данных
- **Автоматизированная отправка** данных на основной сервер AeroToolKit

### Сеть:

- Docker bridge network для межсерверного взаимодействия между AeroToolKit и Photo Server

## Технологический стек

### Backend & Infrastructure

- **Django Backend Architecture** - два независимых сервера с ORM, административной панелью и системой аутентификации
- **Server-Side Rendering** - генерация HTML на стороне сервера с Bootstrap стилизацией
- **REST API Development** - полнофункциональное API с токенной аутентификацией на Django REST Framework
- **Docker Containerization** - изолированная среда развертывания через Docker-контейнеры
- **Data Storage Infrastructure** - надежное хранение в изолированной СУБД PostgreSQL
- **Performance Monitoring** - инструменты отладки с Django Debug Toolbar

### Data Processing & Integration

- **Base64 Image Processing** - преобразование бинарных данных в текстовый формат для безопасной передачи
- **Cross-Server Data Integration** - автоматизированный обмен данными между сервисами с JSON-форматированием
- **Computer Vision Integration** - автоматическое распознавание объектов моделью YOLO перед сохранением
- **Quality Control System** - гибкая настройка параметров распознавания и валидации результатов

### User Management & Analytics

- **User Management System** - система управления сотрудниками с индивидуальной аутентификацией
- **User Activity Dashboard** - отслеживание действий с фильтрацией по пользователям и детализацией загрузок
- **Automated API Documentation** - интерактивная документация через Swagger/OpenAPI

## Быстрый запуск

### Запуск через Docker (рекомендуется)

```bash
# Клонирование репозитория
git clone https://github.com/Alexman2505/Aeroflot_Hackathon.git
cd Aeroflot_Hackathon

# Настройка переменных окружения (обязательно!)
cp backend/.env.example backend/.env
cp photo_server/.env.example photo_server/.env

# Запуск всех сервисов
docker-compose up --build -d
```
